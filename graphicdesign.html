Graphic Design and Digital Illustration

JPEG Illustration

JPEG/JPG is the best and the most common image format for web. In fact, most digital cameras and smartphones save photos as JPEG.

We create digital illustrations on JPEG format.
Contact us for more information.

PNG Illustration

PNG format, or Portable Network Graphics, is a lossless compression file format, which makes it a common choice for use on the Web.

PNG is a good choice for storing line drawings, text, and iconic graphics at a small file size.

SVG Illustration

SVG, or Scalable Vector Graphics , are an XML based markup that can contain two-dimensional vectors. The vectors can be simple shapes, paths, or well just about anything you can do in Illustrator. It's an image format that has more in common with a web page that it does with a JPEG.

We create SVGs! Contact us for more details.
